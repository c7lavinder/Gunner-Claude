# NAH Wholesale Playbook

**New Again Houses — Complete Operational and Training Reference**

---

## What This Is

The NAH Wholesale Playbook is a unified reference package containing four categories of material that work together to support the NAH wholesale real estate operation. It combines team-specific performance profiles, role-specific training materials, industry-level knowledge, and the original source curriculum into a single, organized resource.

This playbook is designed to serve multiple purposes: onboarding new team members, coaching existing staff, feeding the Gunner AI intelligence engine, and providing a structured reference for operational decisions.

---

## Structure

The playbook is organized into three top-level sections, each serving a distinct function.

### 1. User Profiles (`User-Profiles/`) — 3 files

Detailed performance profiles for each core team member, built from actual call grading data, coaching feedback, and performance metrics. Each profile covers what the team member does well, where they need development, their communication style, coaching priorities, and key metrics to track.

## How the Sections Connect

The three sections are designed to work together in a specific flow:

1. **Training Materials** provide the NAH-specific scripts, methodology, and best practices — the "what to say and do" for each role at New Again Houses.
2. **Industry Knowledge** provides the broader wholesale real estate principles and frameworks from five expert coaches — the "why it works" and the theory behind the practice.
3. **User Profiles** apply those principles to actual team members — the "how they're doing it" and "what to improve" based on real call data.

A new Lead Manager, for example, would start with the Training Materials scripts and methodology for their role (files prefixed `LM_`), reference the Industry Knowledge files for deeper understanding (files tagged `LM_` and `ALL_`), and then review their User Profile for personalized coaching priorities.

## File Naming Convention

All files across Training Materials and Industry Knowledge follow a consistent naming pattern:

```
{Role Tag}_{Topic Name}.md
```

Role tags indicate which team roles should prioritize the material:

| Tag | Meaning |
|-----|---------|
| `ALL_` | Applicable to all roles |
| `LG_` | Primary audience: Lead Generators (Cold Callers) |
| `LM_` | Primary audience: Lead Managers |
| `AM_` | Primary audience: Acquisition Managers (Closers) |

## Usage Notes

These materials are formatted in Markdown for maximum compatibility. They can be used directly as reference documents, imported into knowledge management systems, fed into AI training pipelines, or converted to PDF for distribution.

The User Profiles contain performance data current as of the export date. They should be updated periodically as new call grading data becomes available.
