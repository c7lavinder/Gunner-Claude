# Gunner Training Materials

All training materials exported from the Gunner platform (New Again Houses tenant).

## Structure

- **01-Scripts/** — Call scripts for each role (LG, LM, AM)
- **02-Objection-Handling/** — Objection handling guides and rebuttals
- **03-Methodology/** — Sales methodology, techniques, and company info
- **04-Best-Practices/** — Best practices, compliance, CRM guides

## File Naming Convention

Each file is prefixed with the role abbreviation:
- **LG** = Lead Generator
- **LM** = Lead Manager
- **AM** = Acquisition Manager
- **ALL** = Applies to all roles
